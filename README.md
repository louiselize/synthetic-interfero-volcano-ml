# synthetic-interfero-volcano-ml

Generation of synthetic deformation data for volcanoes and ML models.

## Installation

1. Clone this repository:

```bash
git clone https://github.com/louiselize/synthetic-interfero-volcano-ml.git
cd synthetic-interfero-volcano-ml
````

2. Create a virtual environment and install the dependencies:

```bash
python -m venv venv
source venv/bin/activate        # On Windows: venv\Scripts\activate
pip install -r requirements.txt
```

3. Install **SyInterferoPy** in the project folder (not included in this repo):

```bash
git clone https://github.com/matthew-gaddes/SyInterferoPy.git
```
